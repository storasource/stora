# Welcome to Stora

Stora is a comprehensive mobile app store automation platform that streamlines the deployment process for iOS and Android applications. Built for developers and teams, Stora automates the tedious aspects of app store submissions while maintaining quality and compliance.

## What is Stora?

Stora combines AI-powered screenshot automation, intelligent compliance checking, and seamless deployment workflows to help you ship mobile apps faster.

## Key Features

### AI-Powered Screenshots

- **Agentic Exploration**: AI automatically navigates your app and captures interesting screenshots
- **Smart Deduplication**: Avoids capturing duplicate or similar screens
- **Maestro Integration**: Built on the robust Maestro testing framework
- **Cross-Platform**: Works with iOS and Android apps

### Intelligent Compliance

- **Rule-Based Checks**: Validates Info.plist, AndroidManifest.xml, and common issues
- **AI Analysis**: Uses Google Gemini to discover issues that rule-based checks miss
- **Multi-Platform**: Supports iOS, Android, or both
- **Detailed Reporting**: Scores, grades, and prioritized recommendations

## Quick Start

```bash
# Install Stora CLI
npm install -g stora

# Capture screenshots
stora screenshots com.example.app --max-screenshots 10

# Check compliance
stora compliance ./my-app --platform ios
```
